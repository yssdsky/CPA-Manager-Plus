# CPA Manager Plus v1.5.0

> 10 commits · 49 files changed · +6845 / -110

> [English ->](./v1.5.0-en.md)

## Overview

本次发布正式打通 CPA 插件生态。管理面板新增插件管理、插件商店、插件资源页和 OAuth 插件提供商支持，Manager Server 同步补齐插件 API 与资源代理，让插件可以从安装、配置到页面集成完成闭环。

## Highlights

### Features

- 新增插件管理页，支持查看插件状态、启停插件、编辑配置、删除插件，并展示菜单、配置项、作者与版本信息。(`plugins`, `web`)
- 新增插件商店页，支持来源聚合、搜索筛选、安装插件、更新状态识别和安装后刷新。(`plugin-store`, `web`)
- 新增插件资源页和动态侧边栏导航，启用的插件可以把自定义页面嵌入管理面板。(`plugin-pages`, `web`)
- OAuth 登录页支持插件提供商，插件可扩展登录入口并展示独立图标与提示。(`oauth`, `web`)
- 可视化配置新增插件系统开关、插件目录和插件商店来源配置。(`config`, `web`)
- Manager Server 新增 CPA 插件、插件商店和插件资源代理路由，并补齐兼容性测试覆盖。(`manager-server`, `proxy`)

## Upgrade Notes

- 需要连接支持插件 API 的 CPA 版本；旧版 CPA 会在管理面板显示插件功能不受支持。
- 安装、删除、启停或配置部分插件后，可能需要重启 CPA 才能完全生效。
