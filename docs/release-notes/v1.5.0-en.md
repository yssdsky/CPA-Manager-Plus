# CPA Manager Plus v1.5.0

> 10 commits · 49 files changed · +6845 / -110

> [中文 ->](./v1.5.0-zh.md)

## Overview

This release opens the CPA plugin ecosystem inside CPA Manager Plus. The management panel now includes plugin management, a plugin store, embedded plugin resource pages, and plugin-backed OAuth providers, while Manager Server proxies the matching CPA plugin APIs and resources.

## Highlights

### Features

- Added the plugin management page for status review, enable/disable actions, configuration editing, deletion, and plugin metadata display. (`plugins`, `web`)
- Added the plugin store page with source aggregation, search and status filters, plugin installation, update detection, and post-install refresh handling. (`plugin-store`, `web`)
- Added embedded plugin resource pages and dynamic sidebar navigation for enabled plugin menu entries. (`plugin-pages`, `web`)
- Extended OAuth login with plugin providers, including plugin-specific labels, hints, and icons. (`oauth`, `web`)
- Added visual configuration controls for the plugin system toggle, plugin directory, and plugin store sources. (`config`, `web`)
- Added Manager Server proxy routes for CPA plugin, plugin store, and plugin resource endpoints with compatibility test coverage. (`manager-server`, `proxy`)

## Upgrade Notes

- A CPA version with plugin API support is required; older CPA versions will show the plugin feature as unsupported in the management panel.
- Installing, deleting, enabling, disabling, or configuring some plugins may require a CPA restart before the change fully takes effect.
